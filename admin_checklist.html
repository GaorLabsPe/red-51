Checklist de Configuración para Administradores de Labs51

Sigue estos pasos para asegurarte de que tu tienda esté completamente configurada y lista para vender.

  • Abre la tienda por primera vez e ingresa la URL de tu proyecto y la clave pública (anon key) de Supabase.
  • Ve al panel de Admin → Configuración. Copia el script SQL y ejecútalo en el Editor SQL de tu proyecto de Supabase para crear todas las tablas necesarias.
  • En tu panel de Supabase, ve a Authentication → Users y crea las cuentas para los administradores que gestionarán la tienda.
  • En Admin → Config

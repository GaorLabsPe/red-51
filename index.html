         Cambios Crticos (Obligatorios)
        
                                        
        
                                            
        1. Evolution API (EN TODOS LOS NODOS DE WHATSAPP)
        
                                            
        Nodos afectados: Enviar WhatsApp Admin, Confirmacin al Cliente, etc.
        
                                            
        Qu cambiar: URL de tu API, API Key y seleccionar tus credenciales.
        
                                        
        
                                        
        
                                            
        2. Nmero de WhatsApp del Administrador
        
                                            
        Nodo: Enviar WhatsApp Admin
        
                                            
        Qu cambiar: En Body Parameters  number, reemplazar 51905820448 por el nmero del admin.
        
                                        
        
                                         
        
                                            
        3. Datos de Pago (EN 5 LUGARES)
        
                                            
        Nodos: Formatear Mensaje, Mensaje 6h, Mensaje 24h, Mensaje 48h.
        
                                            
        Qu cambiar: Reemplazar el nmero de Yape/Plin 975615244 y agregar otras cuentas bancarias.
        
                                        
        
                                        
        
                                            
        4. URLs de Webhooks y Credenciales Supabase
        
                                             
        Asegrate de copiar las URLs de los webhooks a Vercel y seleccionar las credenciales de Supabase en los nodos correspondientes.
        
                                        
        
                                         
        
                                            
        5. URL de la Tienda
        
                                            
        Nodo: Preparar Mensaje de Cancelacin
        
                                            
        Qu cambiar: Reemplazar la URL por defecto https://red-51.vercel.app/ por la URL de la nueva tienda.
